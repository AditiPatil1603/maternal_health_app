 Maternal Health App

A Machine Learning-powered web application designed to predict maternal health risks and assist in early intervention. This project aims to improve healthcare outcomes for pregnant women by analyzing health parameters and providing risk predictions.

---

 Features

-  Predict maternal health risk levels (Low, Medium, High)
-  Machine Learning model integration
-  User-friendly web interface
-  Organized project structure for deployment
-  Fast and efficient predictions
